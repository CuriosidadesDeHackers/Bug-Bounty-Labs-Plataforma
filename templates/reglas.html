Reglas para la Creación de Laboratorios

Guía completa para crear y enviar laboratorios a Bug Bounty Labs

Formatos Aceptados

  • Docker o Máquina Virtual: Tienes 2 formas de poder crear tu laboratorio: mediante Docker o máquina virtual (OVA).
  • Compatibilidad: Si creas tu laboratorio como máquina virtual, debe funcionar en VirtualBox (Vmware solo de forma excepcional).

Requisitos Técnicos

  • Tamaño: No hay límite de GB para los laboratorios, pero si es sencillo, se recomienda usar sistemas operativos ligeros como Debian.
  • Dominio: No es obligatorio que el laboratorio tenga un dominio configurado, pero si lo tiene, debe usar la extensión .bbl.
  • Subdominios: El laboratorio puede contener subdominios.

Contenido y Enfoque

  • Vulnerabilidad Específica: El laboratorio debe enfocarse exclusivamente en una vulnerabilidad web.
  • Documentación: Debe incluir una guía clara de instalación y configuración.
